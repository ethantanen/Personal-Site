Programming Experience

WWTracker

WWTracker is an application I created to send whitewater kayakers information on whitewater conditions in Colorado. The application was written using Node.js with an Express.js framework. The program utilizes the Twilio API for SMS capabilities, the DarkSky API for weather information and the USGS API for information on river flows. The program is hosted on Herkoku

Neural Network

Neural Network is an application that uses a feedforward neural network to recognize handwritten digits. The program was coded in C with use of OpenMP to run portions of the training phase in parallel on Colorado College’s cluster. The NN was trained on the MNIST dataset.

Pickup Game Organizer

Pickup Game Organizer is an application that allows students at Colorado College to schedule pickup sports games around campus. My role in the team was to create and manage the app’s database using SQL and phpMyAdmin.
